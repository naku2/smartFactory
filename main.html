Transistor/Defective/Detection

This page was produced under the theme of 'Development of Automatic Detection System for Production Line Defective Products Using Computer Vision'. The purpose of this project is to develop a system that automatically detects defective products on the production line using computer vision technology. Rapid detection and classification of defective products during production is critical to improving production efficiency and ensuring product quality.

Find good or bad

The target of detection I selected is a transistor, and the types of defects include bent_lead, cut_lead, damaged_case, and misplaced. Although there are various types of defects, the dataset was so insufficient that we decided to combine all of these defects into 'bad'. The model I used was 'ResNet50', which shows a low val_loss, but the value is fluid, and I am not sure why.
